MR火星车学习笔记（1）——初步配置及上手

*初步配置及上手*

*下载插件和包*

从 Microsoft 下载中心下载混合现实功能工具的最新版本。在混合现实功能工具中，选择“启动”。选择“浏览”按钮（它是下图中的“三个点”按钮），导航到你的项目，然后将其打开。

检验路径没问题后，选择“发现功能”。单击“混合现实工具包(0/10)”左侧的“+”按钮，然后选择“混合现实工具包基础版”的最新版本（foundation）。单击“平台支持(0/5)”左侧的“+”按钮，然后选择“混合现实 OpenXR 插件”的最新版本。

做出选择后，单击“获取功能”。选择“验证”以验证所选的包。 显示“未检测到任何验证问题”后，单击“确定”

*配置unity*

系统应显示 MRTK 项目配置器。如果未显示，手动打开它：在菜单栏上，选择“混合现实”>“工具包”>“实用程序”>“为 MRTK 配置项目”：

  1. 选择“Unity OpenXR 插件”以启用 XR 插件管理，并将 Unity OpenXR 插件添加到项目。
  2. 在“欢迎使用 MRTK!”屏幕上，选择“显示 XR 插件管理设置”。
  3. 随即打开“项目设置”窗口。确保你位于显示“通用 Windows 平台”设置（Windows 徽标选项卡）的“XR 插件管理”页面上。（unity要事先安装universal windows platform build support模块）
  4. 确保选中“在启动时初始化 XR”，然后在“插件提供程序”下单击“openXR”。
  5. 加载 OpenXR 插件，然后在“OpenXR”下方出现两个项目。 选择第一个，即“Microsoft HoloLens 功能组”。
  6. 在“MRTK 项目配置器”窗口中，选择“应用设置”。
  7. 单击“下一步”，然后单击“导入 TMP Essentials”。
  8. 出现“MRTK 安装程序已完成!”面板时，单击“完成”。

为 OpenXR 配置项目设置：

1．Unity 重启后，在菜单栏中，选择“编辑”>“项目设置…”。（project settings）

2．在“项目设置”窗口的左侧列中，选择“播放器”。（player）

3．单击“发布设置”（publishing settings）下拉列表，然后在“包名称”字段中输入合适的名称，例如 MRTK-Tutorials-Getting-Started。（包名称是应用的唯一标识符。 如果要避免覆盖以前安装的同名应用版本，则应在部署应用之前更改此标识符。）

在菜单栏中，依次选择“混合现实”>“工具包”>“添加到场景并配置…”。（“混合现实工具包”设置会显示在检查器中）

*Unity 编辑器内输入模拟*

你可以使用 Unity 的编辑器内输入模拟功能来测试全息对象行为。

*更改场景中的视图：*

要向前/向左/向后/向右移动摄像机，按 W/A/S/D 键。

要垂直移动相机，按 Q 和 E 键。

要旋转相机，按鼠标右键，然后拖动。

*模拟手控输入：*

要启用模拟右手，按住空格键。 要移除该手，松开空格键。

要启用模拟左手，按住左 Shift 键。 要移除该手，松开该键。

要在场景中移动左手或右手，移动鼠标。

要向前或向后移动手，滚动鼠标滚轮。

要模拟收缩手势，单击鼠标左键。

要旋转手，按住空格键 + CTRL 键（右手）或按住左 Shift 键 + CTRL 键（左手），然后移动鼠标。

*持久的手*

要在不继续按住某个键的情况下启用手并使它在屏幕上，请按 T（左手）或 Y（右手）。 再次按这些键可移除手。

*Unity 中生成应用程序*

在菜单栏中，选择“File”>“Build Settings…”。

在“生成设置”窗口中，单击“添加打开的场景”按钮，将当前场景添加到“生成中的场景”列表中 。单击“生成”按钮。

在“Build Universal Windows Platform”窗口中，导航到希望用于存储生成的文件夹，或者新建一个文件夹并导航到它，然后单击“Select Folder”按钮开始生成过程。

*Unity中与模型交互*

若要使对象在使用跟踪手时可抓取，必须附加三个组件：

碰撞体组件（无需在此执行任何操作 - Unity 立方体默认已附加盒型碰撞体）

“对象操控器(脚本) objectManipulator”组件

NearInteractionGrabbable（脚本）组件
